About These Gauges

Each gauge displays the rate of change (in degrees 0-90°) for VM resource metrics. The angle is computed as: atan((newValue - oldValue) / timeDelta) * 180 / π

This shows how fast each metric is changing, providing insight into VM resource activity and trends.
